## Installation
#### Clone Repo
```git clone https://github.com/Arsh90/dmenu-nord```
#### Cd into the folder
<code>cd dmenu-nord</code>
#### Install It
<code>sudo make clean install</code>
#### Run It
```dmenu_run```
